# Challenge 2 - Play Fetch Unity Project

This repository contains a small Unity project created for the "Challenge 2" tutorial series. It demonstrates basic scripting concepts such as object spawning, movement, collision detection and simple game logic. The scene shows balls falling from the top of the screen while the player can launch dogs to intercept them.

> **Note:** This project folder came from a Unity tutorial, so many of the assets/scripts are minimal and meant to teach beginners.

## ðŸš€ Getting Started

### Prerequisites

- Unity Editor (2020.3 LTS or later recommended)
- Git (if you plan to clone the repository)

### Opening the project

1. Clone or download this repository to your machine.
2. Launch the Unity Editor and choose **Open**.
3. Browse to the folder containing `Assets`, `ProjectSettings`, etc. and open it.
4. Allow Unity to import assets; this may take a few minutes on first load.

### Running the scene

- Open the scene `Assets/Scenes/Challenge 2.unity` from the **Project** window.
- Press the **Play** button in the Editor to start the game.
- Press **Space** during play mode to spawn a dog and try to hit the falling balls.
